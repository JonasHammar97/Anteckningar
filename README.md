# Anteckningar

Vår gemensamma anteckningstjottahejt.

1. Skapa din egna branch med ditt namn.
2. Skriv dina anteckningar, commita, pusha och vouila.

Vill du slå ihop dina anteckningar med någon annans så använd "merge branches".
